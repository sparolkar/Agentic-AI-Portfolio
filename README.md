# Agentic-AI-Portfolio
AI Agents: Building single/multi-agent and multi-modal workflows using LLM, RAG, and MCP Servers with low code / no code platforms (Zapier, n8n, Langflow, Langfuse, ElevenLabs).
Selected projects:
o	Incident Management – Triages and assigns incidents to specific teams (Zapier): https://agents.zapier.com/copy/948885be-9b6f-462c-b9d1-8783f490e535
o	Product Launch Readiness – Synthesizes inputs from engineering, legal, and marketing (Zapier): https://agents.zapier.com/copy/b2296799-8cb4-40ec-bb5a-75353f97d620
o	Checkout Incident Detection – Rule-based detection and alerting (Zapier): https://agents.zapier.com/copy/456466b8-678b-4e86-864e-c31dfdaba969
o	Product Finance Reporting – RAG-powered financial reporting (n8n)
o	Project Manager’s Daily Highlights – Multi-agent workflow aggregating Slack, email, and calendar (n8n)
o	Project Status Chatbot – Slack-integrated chatbot over project documents using multi-agent workflow (n8n)
o	Project Health Monitor – Voice/text input multi-agent workflow (ElevenLabs)
o	Feature Prioritization – Multi-agent workflow (Langflow)
o	LLM Evaluation & Fine-Tuning – Evaluation pipeline using Langfuse and LLM-as-a-judge methodology
o	Calendar Mate – Daily Briefing, Smart Scheduling, and Email Summarization (n8n)
o	Sales Genie – Lead Qualification, Product Recommendations, and Weekly Sales Insights (n8n)
o	PRD Genie – Requirement Extraction, PRD Generation, and Epic/User Story Breakdown (n8n)
o	Project Manager Assistant – Project Plan Generation, Risk Assessment, and Weekly Status Report (n8n)
